# 日程調整アシスタント 使い方ガイド

LINE の横に置いて使う、スマホ向けの日程調整ツールです。

**公開 URL:** https://line-yoga-system.streamlit.app

スマホで開き、**共有 → ホーム画面に追加** するとアプリのように使えます。

---

## 1. 基本的な流れ

```
空き枠を探す → LINE で相手に送る → 仮押さえ → 確定
```

1. **空き枠の探し方** で所要時間などを設定
2. 表示された候補を **まとめてコピー** して LINE に貼る
3. 相手に送った枠を **仮押さえキープ**
4. 返事が来たら **確定**（TimeTree 登録は今後対応）

---

## 2. 空き枠の探し方

| 設定 | 説明 |
|------|------|
| 所要時間 | 1時間など、面談に必要な長さ |
| 開始刻み | 候補を何分おきに出すか |
| 前後のバッファ | 前後に空けたい時間 |
| 何日先まで | 提示する日数の範囲 |
| 営業時間 | この時間帯だけ候補にする |

> スクリーンショット: 「🔍 空き枠の探し方」を開いた画面

---

## 3. LINE に送る文面

- **候補をまとめてコピー** … 定型文付きで一括コピー
- **✏️ 送信文を編集** … 自分の言い回しに変更
- **編集完了** … 言い回しを保存（次回以降も使われます）
- **定型文に戻す** … 初期の文面に戻す

> スクリーンショット: 「LINE用にまとめて送る」エリア

---

## 4. 提示しない時間（プライベート）

TimeTree に予定がなくても、社外には出したくない時間を除外できます。

- **毎週土日を非表示** / **平日18時以降を非表示**
- **カスタムルール** … 曜日・時間帯を指定
- **この日だけ非表示** … 実家に帰る日など

> スクリーンショット: 「🚫 提示しない時間」エリア

---

## 5. 仮押さえ（返事待ち）

相手の名前と日時を選んで **仮押さえキープ** します。

- **今日キープ** / **3日前にキープ** など経過日数を表示
- **3日以上** 経過すると赤く表示（フォローしやすくなります）
- **確定** … 最終確認のあと登録
- **解除** … キープを取り消し

> スクリーンショット: 「仮押さえ中のリスト」

---

## 6. データの保存について

画面上部に保存先が表示されます。

| 表示 | 意味 |
|------|------|
| スプレッドシートに保存中 | クラウドでもデータが残る（推奨） |
| このブラウザのセッション内のみ | ブラウザを閉じると消える |
| この Mac に保存中 | ローカル利用時 |

### スプレッドシート連携（永続保存・推奨）

ブラウザを閉じても仮押さえ・ルール・送信文が残ります。

1. Google スプレッドシートを新規作成
2. サービスアカウントを作成（Google Cloud）
3. スプレッドシートをサービスアカウントのメールと「編集者」で共有
4. Streamlit Cloud の Secrets に設定（`SHEETS_SETUP.md` 参照）

---

## 7. 困ったとき

| 症状 | 対処 |
|------|------|
| ページが開けない | Streamlit のサーバーが止まっている。URL を再確認 |
| 空き枠が出ない | 所要時間・営業時間・バッファを緩める |
| データが消えた | スプレッドシート連携を設定する |
